    Stochastic Processes

    A stochastic process is a mathematical tool for modeling time-dependent random phenomena. While a single random variable represents a static outcome, a stochastic process \((X_t)_{t \in T}\) describes a system that evolves over a set of time indices T.

    Standard Random Walk

    A simple unrestricted random walk \((S_n)_{n \geq 0}\) starting at \(S_0 = 0\) is defined by:

    $$ S_n = X_1 + X_2 + ... + X_0 = \sum_{k=1}^{n} X_k $$

    where:

    • \(S_n\): value or position of the process at instant \(n\)
    • \(X_k\): specific movement performed at step \(k\)

    The Markov Property

    A process is Markov when its statistical behavior after time \(t\) can be recovered from the value \(X_t\) of the process at time \(t\). In particular, the values \(X_s\) of the process at time \(s \in [0,t)\) have no influence whatsoever on this behavior as long as the value of \(x_t\) is known.

    \[ P(S_{n+1}=j \mid S_n=i_n,\ldots,S_1=i_1) = P(S_{n+1}=j \mid S_n=i_n) \]

    Control Approaches

    We could consider three approaches to control the system:

    • One single control law: The differences in the dynamic behavior due to the changes in the operation mode are treated as disturbances. This would be a kind of Standard Robust Control approach.
    • Two control laws: Each of the control laws is independently designed and changes when the operation mode changes. This could be called Multimodel approach.
    • Two control laws: They are designed considering the Markov nature of the jumps between operation modes. This could be called Markov Jump approach.

    Robust Control Approach - Issues:

       

    • One would expect poorer performance, especially when system dynamics for each operation mode differ significantly.
    • Also, due to the random switching between operation modes, in a time-variant system, the stability guarantees of most of the standard design techniques can be compromised.

    Multimodel Control Approach - Issues:

       

    • On a first analysis, present better performance.
    • Stability issues still remain.
    • Question:Would the best controllers for each operation mode when associated, result in the best controller for the overall system?

    Not Necessarily!

    Markov Jump Approach - Issues

       

    • For a specific unfavorable sequence of operation modes, the performance may be very poor, but in the long run, or for a great number of sequences, it is likely to present the best results.
