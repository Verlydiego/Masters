Stochastic Processes

A stochastic process is a mathematical tool for modeling time-dependent random phenomena.

\[ (X_t)_{t\in T} \]
Represents a system that evolves through time.

Standard Random Walk

A simple unrestricted random walk starting at \(S_0=0\):

\[ S_n = \sum_{k=1}^{n} X_k \]
  • \(S_n\): position at instant \(n\)
  • \(X_k\): movement performed at step \(k\)

The Markov Property

The future depends only on the present state.

\[ P(S_{n+1}=j \mid S_n=i_n,\ldots,S_1=i_1) = P(S_{n+1}=j \mid S_n=i_n) \]

The past becomes irrelevant once the current state is known.

Markov Jump Linear Systems

Consider a set of dynamical systems:

\[ \{G_1,G_2,\ldots,G_N\} \]

The system switches among these operation modes.

MJLS Concept

The switching process follows a Markov chain.

\[ \theta(k)\in\{1,2,\ldots,N\} \]
\(\theta(k)\) identifies the current operation mode.

Transition Probability Matrix

The jumps are characterized by:

\[ P= \begin{bmatrix} p_{11}&p_{12}&\cdots&p_{1N}\\ p_{21}&p_{22}&\cdots&p_{2N}\\ \vdots&\vdots&\ddots&\vdots\\ p_{N1}&p_{N2}&\cdots&p_{NN} \end{bmatrix} \]

Each row sums to one.

Operation Mode

At each instant:

\[ \theta(k)=i \]

means the system is operating in mode \(G_i\).

Throughout this course, we assume \(\theta(k)\) is known.

Example: Two Operation Modes

\[ G_1:\quad x(k+1)=0.8x(k) \]
\[ G_2:\quad x(k+1)=1.2x(k) \]

Transition Matrix

\[ P= \begin{bmatrix} 0.7 & 0.3\\ 0.4 & 0.6 \end{bmatrix} \]

Assume \(\theta(0)=1\).

Possible Realizations

For a horizon of 20 steps:

\[ 2^{20}=1\,048\,576 \]

Different Markov-chain realizations produce different trajectories.

Control Approaches

  1. Robust Control Approach
  2. Multimodel Approach
  3. Markov Jump Approach

Robust Control

  • Single control law
  • Mode variations treated as disturbances
  • May lead to conservative designs
  • Performance degradation is expected

Multimodel Control

  • One controller per operation mode
  • Usually better performance
  • Stability issues remain
Question:

Are the best individual controllers also the best overall controller?

Answer

Not Necessarily!

Markov Jump Approach

  • Explicitly considers transition probabilities
  • Uses stochastic information
  • Usually provides better long-term performance

General Dynamic System

\[ \begin{aligned} x(k+1)&=A_{\theta(k)}x(k) +B_{\theta(k)}u(k) +G_{\theta(k)}w(k) \\[4mm] y(k)&=L_{\theta(k)}x(k) +H_{\theta(k)}w(k) \\[4mm] z(k)&=C_{\theta(k)}x(k) +D_{\theta(k)}u(k) \end{aligned} \]
